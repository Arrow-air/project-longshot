# build123d CAD

Parametric CAD source for Longshot build123d models.

## Suggested layout

- `src/` — Python source for models and assemblies.
- `exports/step/` — generated STEP files.
- `exports/stl/` — generated STL files.
- `reference/` — local reference geometry used by build scripts.

Add setup and export commands here once the first build123d model is committed.
