# Electronics

Longshot electronics design files live here.

## PCB projects

Each PCB should have its own self-contained folder under `pcbs/`. KiCad projects should keep project files under `kicad/` and local libraries under `kicad/libs/` so a contributor can clone/download the PCB folder and open it without missing project-specific components.

Current PCB folders:

- `pcbs/BMSJ/` — Longshot BMSJ battery management system board.
- `pcbs/BMSZ/` — Longshot BMSZ battery management system board.
