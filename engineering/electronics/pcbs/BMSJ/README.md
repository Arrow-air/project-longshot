# BMSJ

KiCad PCB project for the Longshot BMSJ battery management system board.

## Layout

- `docs/` — requirements, architecture notes, connector pinouts, and design decisions.
- `firmware/` — board-specific firmware or configuration artifacts, if needed.
- `images/` — renders, diagrams, and review screenshots.
- `kicad/` — KiCad project, schematic, PCB, and project-local libraries.
- `manufacturing/` — generated fabrication outputs and assembly packages.

## KiCad

Open `kicad/BMSJ.kicad_pro` in KiCad. Project-local symbols, footprints, and 3D models live under `kicad/libs/`.
