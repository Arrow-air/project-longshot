# Source Import

SL_PCB was imported from [`Julius-eng/Vector-BMS`](https://github.com/Julius-eng/Vector-BMS) and adapted into the Project Longshot engineering layout.

Imported:

- KiCad project, schematic sheets, and PCB as `kicad/SL_PCB.*`
- VESC BMS hardware configuration under `firmware/vesc_bms_hw/`, renamed to `hw_sl_pcb.*`
- Project footprint(s) under `kicad/libs/SL_PCB.pretty/`
- Source README content, adapted for the Longshot folder structure

Not imported:

- KiCad lock/cache files
- KiCad auto-backup zip files
